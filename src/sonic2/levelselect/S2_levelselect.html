The level select text in Sonic 2 is part of the level select's plane mappings, with no straightforward method to edit it through source code text alone.

I recommend you use MainMemory's SonPLN, which comes with a handy text editor for the level select text out of the box. The required SonPLN.ini project file should already be in your disassembly's folder — if not, you can download it here.
